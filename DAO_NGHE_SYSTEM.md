# Đạo Nghiệp · 10 nghề code-tu tiên

Đạo Nghiệp là hệ nghề nghiệp chính thức, không phải nghề phụ. Mỗi người chọn một nghề chính, có cấp nghề, đạo tài riêng, làm nghề mỗi ngày và chế tạo vật phẩm theo nghề.

## 10 nghề

| Nghề | Vai trò |
|---|---|
| Debug Đạo Sư | Trị bug, giảm tâm ma, khắc Bug Ma |
| Core Forge Sư | Rèn đạo khí/core, tăng lực chiến |
| Runtime Trận Sư | Trận pháp runtime, buff party/bí cảnh |
| Patch Lệnh Sư | Hotfix, rollback, cứu nguy |
| Data Mạch Sư | Khai schema địa mạch, vật phẩm database |
| Cloud Pháp Sư | Container/cloud pháp đàn, tăng hộ thể |
| Security Hộ Đạo Sư | Firewall, zero trust, phòng thủ |
| AI Diễn Toán Sư | Prompt/vector/diễn toán, tăng chí mạng |
| Frontend Huyễn Diện Sư | DOM/CSS/huyễn diện, tăng chính xác/tốc |
| Git Mệnh Sư | Branch/merge/commit nhân quả, hồi khí/cống hiến |

## Gameplay loop

Chọn nghề → làm nghề hằng ngày → nhận tinh thông + đạo tài → chế tạo vật phẩm → dùng/trang bị/tặng/hiến tông.

## Cân bằng

- Làm nghề 1 lần/ngày.
- Chế tạo 1 lần/ngày.
- Đổi nghề mất cống hiến nếu đã có tiến độ.
- Vật phẩm nghề vẫn dùng hệ phẩm chất cũ.
